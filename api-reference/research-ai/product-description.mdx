title: "Research AI"
description: "Chat with Statista Research AI"
---

The Research AI API is intended for users who want to use or embed Statista's Research
AI. (Try it <a href="https://www.statista.com/research-ai/">here</a>)

In order to use this API; you'll need to POST a request (question) and poll the results
using a GET request.

The response from the GET request will contain:

- Generated Answer
- Key facts
- Sources
