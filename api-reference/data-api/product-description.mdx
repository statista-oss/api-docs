---
title: "Data API"
description: "Fetch detailed data on a specific statistic by ID"
---

The Data API is intended for users who want in-depth data and chart values for a specific
statistic. That ID can be found either with the Discovery API or directly on Statista's
website.

#### Endpoints

The `Data API` package allows access to the following endpoints:

- [Statistic](/api-reference/data-api/get-statistic)
- [Report](/api-reference/data-api/get-reports)

The `Data API` package also includes endpoints similar to those in the `Discovery Advanced` package:

- [Statistics](/api-reference/discovery-advanced/get-statistics) + [Search Statistics](/api-reference/discovery-advanced/get-search-statistics)
- [Infographics](/api-reference/discovery-advanced/get-infographics)
- [Studies](/api-reference/discovery-advanced/get-studies)
- [Market Insights](/api-reference/discovery-advanced/get-marketInsights)
