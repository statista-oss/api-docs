## OpenAI Platform
To try the Statista MCP Server with OpenAI, follow these steps:
1. Go to [platform.openai.com](https://platform.openai.com)
2. If you don't already have one; create an account
3. Go to **Dashboard** -> **Chat**
4. Click **Tools (+ Add)**
5. Select **MCP Server**
6. Add provided URL and label
7. Under **Authentication**, change to **Custom Headers**, set **header** to **x-api-key** and enter provided [API Key](/start/request-api-key)
