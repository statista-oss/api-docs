## Install Perplexity Desktop
To try the Statista MCP Server with Claude Desktop, follow these steps:
1. Install [Perplexity Desktop](https://www.perplexity.ai/platforms) (important that it's the desktop version)
2. If you don't already have one; create an account
3. Open Perplexity Desktop

## Set up Connector
1. In the bottom-right corner, click cogwheel
<Frame>
<img 
  src="/images/perplexity-settings.png" 
  alt="Open settings"
  height="100"
  width="400"
  className="rounded-lg"
/>
</Frame>

2. Click `Connectors`
3. Click `+ Add Connector`
4. Choose `Advanced`
<Frame>
<img 
  src="/images/perplexity-create-connector.png" 
  alt="Configure connector"
  height="100"
  width="400"
  className="rounded-lg"
/>
</Frame>

5. Copy/paste this into text-box:
```json
{
  "args" : [
    "-y",
    "mcp-remote",
    <STATISTA_MCP_URL>,
    "--transport",
    "http-only",
    "--header",
    "x-api-key: \\${API_KEY}"
  ],
  "command" : "npx",
  "env" : {
    "API_KEY" : <STATISTA_API_KEY>
  },
  "useBuiltInNode" : true
}
```
<Info>
Replace `<STATISTA_MCP_URL>` and `<STATISTA_API_KEY>` with provided credentials from Statista.
</Info>

6. Save connector 
7. Toggle `Statista MCP` as a source
<Frame>
<img 
  src="/images/perplexity-tools.png" 
  alt="Toggle connector"
  height="100"
  width="400"
  className="rounded-lg"
/>
</Frame>
